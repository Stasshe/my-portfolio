## Overview

体感速度を最優先に設計したリアルタイムチャットアプリです。

メッセージ同期だけでなく、通話、通知、ファイル共有まで含めて、日常利用できる会話基盤を目指しました。

## Objective

- 遅延の少ないチャット体験を実現
- モバイルでも通知導線を維持
- 連絡から共有までをアプリ内で完結

## Architecture Highlights

- Firestore中心のリアルタイム同期
- 複数サーバー構成で負荷分散
- PWA + Service Workerによる通知
- SkyWay(WebRTC)によるビデオ通話
- Google Drive APIによるファイル共有

## Tech Stack

- Firestore
- JavaScript
- PWA / Service Worker
- Firebase Cloud Messaging
- Google Drive API
- WebRTC

## Trade-offs

- 無料基盤の制約によりセキュリティ強化は段階的
- 開発速度とUXを優先した設計判断を含む
- 高機密データの送受信は想定外

## Result

同期速度と機能幅のバランスを取り、実利用できるチャット体験を構築できました。通知導線と通話連携が特に有効でした。
