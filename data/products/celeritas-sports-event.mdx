## Overview

学校のスポーツ祭運営の課題を解決する総合Webシステム。リアルタイムスコア更新、スケジュール管理、トーナメント表示をFirebase Realtime Databaseで実現。生徒会と一般生徒の双方にメリットを提供。

## Tech Stack

- React v18
- TypeScript
- MUI Material
- Firebase Realtime Database
- Firebase Authentication
- g-loot/react-tournament-brackets

## Key Features

- **リアルタイムスコア管理**: 全デバイスでのライブスコア更新 / トーナメント表の可視化
- **行事・スケジュール管理**: 複数のスポーツ行事の作成・管理 / 自動スケジュール生成
- **運営スタッフ機能**: 集中管理ダッシュボード / 行事全体のライフサイクル管理
- **生徒向け機能**: 全生徒向け閲覧専用アクセス / 暑い中試合場所に行く必要なし
