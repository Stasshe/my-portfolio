## Overview

ブックマークレット制作でつまずきやすい「整形」「管理」「再編集」を一つのフローにまとめたWebアプリです。

コードを書ける人向けだけではなく、UI中心で扱いたいユーザーにも使いやすい体験を目指しました。

## Problem

- コードを1行化する段階で失敗しやすい
- 端末をまたぐとスニペット管理が煩雑になる
- 実行前に簡単に確認できる場が欲しい

## What It Solves

- スニペットの作成から保存までを一画面で完結
- ターゲットページを意識したライブプレビュー
- 最適化済みブックマークレットのエクスポート
- タグ・カテゴリによる整理

## Tech Stack

- React
- TypeScript
- Next.js
- Vercel

## Implementation Notes

- 反復編集しやすいデータ管理を重視
- 操作回数を減らすために導線を短く設計
- 視覚的な編集とコードベース編集の両方に配慮

## Result

「作る→試す→直す→保存する」を短サイクルで回せるツールとして機能しました。小規模ユーティリティの量産に向いています。
