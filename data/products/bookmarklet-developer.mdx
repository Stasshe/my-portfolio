## Overview

モバイルブラウザでも、ページ構造の確認や簡単なコード検証を行えるようにした開発用ブックマークレットです。

PC向けDevToolsを完全再現するのではなく、外出先で必要になる最小限の操作を高速に実行できることを重視しました。

## Why This Exists

- iPadやスマホでは要素調査がしづらい
- 「少しだけ確認したい」作業のためにPCを開くのは重い
- ブックマークレットなら導入と共有が速い

## Core Features

- 表示中ページのHTML要素を確認
- 一部DOMをその場で書き換えて検証
- 軽量コンソールでJavaScriptを試行
- `console.log` が難しい環境向けに `alert` ベースの出力も想定

## Tech Stack

- JavaScript
- Bookmarklet
- DOM API
- Mobile Browser

## Implementation Notes

- 配布しやすい1本のブックマークレットとして設計
- 実行時の副作用を抑えるため、処理を局所化
- 複雑なUIは持たず、検証操作に集中した構成

## Trade-offs

- 高度なネットワーク解析や詳細トレースは非対応
- CSPや環境制限によって機能が制約される場合がある
