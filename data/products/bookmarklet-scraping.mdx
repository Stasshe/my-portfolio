## Overview

閲覧制限のある環境で、必要なWeb情報を取得するために作成したブックマークレットです。

独立サイトを作るのではなく、既存ページ上で動く軽量ツールとして設計することで、導入ハードルを下げています。

## Goals

- インストール不要で実行できること
- 制限環境でも最低限の情報取得を可能にすること
- DOM解析結果を短時間で確認できること

## How It Works

- `api.allorigins.win` 経由で対象ページを取得
- 取得コンテンツを解析して必要情報を抽出
- 閲覧しやすい形式へ整形して表示

## Tech Stack

- JavaScript
- DOM Scraping
- Data Parsing
- Bookmarklet
- Proxy API (`allorigins`)

## Caveats

- 対象サイトのDOM構造変更に弱い
- API制限や通信制約により失敗するケースがある
- 利用規約への配慮が前提

## Result

重いセットアップなしで使える「情報取得の初手」として有効でした。短時間の確認用途で特に強みがあります。
