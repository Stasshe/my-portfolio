## Overview

研究モード、AI対局、オンライン対戦・観戦を1アプリに統合した将棋アプリです。

用途ごとに別アプリへ切り替える手間をなくし、学習と実戦を同じ体験で行える設計を目指しました。

## Product Vision

- 研究と対局をシームレスに往復できる
- 保存と再開が自然にできる
- 初心者から中級者まで使える導線を作る

## Major Features

- 研究モードで棋譜保存と駒配置切替
- GIKOU API連携のAI対局
- ルーム作成型オンライン対戦
- チャット連動の参加導線
- 観戦モードと盤面反転
- アプリ再起動後の盤面復元

## Tech Stack

- Swift
- SpriteKit
- Firebase
- WebSocket
- GIKOU AI API
- UIKit

## Engineering Challenges

- 複数モード統合による状態管理の複雑化
- オンライン同期と研究機能の競合解消
- API制約由来の不安定挙動の調整

## Result

機能集約度の高い将棋アプリとして成立し、研究用途と対戦用途の両方で使い分けできる構成に仕上がりました。
