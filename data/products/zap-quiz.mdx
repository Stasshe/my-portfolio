## Overview

Zap!-Quizは、リアルタイム対戦の緊張感と、学習用途の体系性を両立させるために設計したオンラインクイズアプリです。

公式問題とユーザー作成問題の二層構造を持ち、イベント利用から個人学習まで幅広い運用を想定しています。

## Product Goals

- 早押し対戦のテンポをWeb上で再現
- 学習向けの体系的コンテンツを提供
- 問題作成と共有を容易にする

## Core Mechanics

- 最速回答者判定を行う早押しシステム
- ルームリーダーモデルによる進行安定化
- Firestoreでのリアルタイム同期
- 早押し / 選択 / 記述の複数形式
- 表記ゆれを吸収する複数正答設定

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Firebase (Firestore, Authentication)
- Zustand

## Implementation Notes

- 対戦進行の一貫性を最優先に状態設計
- 短時間で判断しやすいUI密度へ調整
- 数式・構造化データ対応で問題表現を拡張

## Result

対戦性と学習性を同時に成立させることができ、継続利用しやすいクイズ基盤として仕上がりました。
