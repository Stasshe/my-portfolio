## README Notes

READMEベースの運用手順は、Codespaces + Docker + noVNCを一直線で構築する形です。

1. Codespacesを起動
2. Seleniumコンテナを起動して4444/7900を公開
3. Seleniumをインストールしてスクリプト実行
4. localhost:7900へ接続してブラウザを操作

運用時はセッションタイムアウトと描画品質(Quality/Compression)の調整が重要です。
